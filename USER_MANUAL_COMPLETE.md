## Getting Started

### Login/Signup
- **Location**: Opens automatically on first visit
- **Features**: Email/password login, biometric login (Face ID/Touch ID on mobile)
- **Forgot Password**: Click "Forgot Password?" link

### First Time Setup
1. Complete profile setup
2. Choose subscription tier (Free, Pro, Team, Enterprise)
3. Optional: Set up team and invite members

## Dashboard

### Location
Main screen after login - shows overview of your inventory

### Features
- **Total Items**: Count of all inventory items
- **Total Value**: Sum of all item values
- **Low Stock Alerts**: Items below threshold
- **Recent Activity**: Latest additions/changes
- **Category Cards**: Quick access to each category (click to filter)

### Quick Actions
- **Add Item**: Blue "+" button (top right)
- **Search**: Search bar at top
- **Filter**: Filter icon next to search
- **Scan Barcode**: Camera icon (mobile)

## Managing Inventory

### 11 Categories
1. **Firearms**: Rifles, pistols, shotguns
2. **Ammunition**: Factory loaded ammo
3. **Optics**: Scopes, red dots, magnifiers
4. **Magazines**: All magazine types
5. **Accessories**: Grips, stocks, lights, lasers
6. **Suppressors**: Sound suppressors/silencers
7. **Cases**: Storage and transport cases
8. **Powder**: Reloading powder
9. **Primers**: Reloading primers
10. **Bullets**: Projectiles for reloading
11. **Reloading**: Brass, dies, equipment

### Adding Items

#### Method 1: Manual Entry
1. Click **"+ Add Item"** button (top right)
2. Select category from dropdown
3. Fill in required fields (marked with *)
4. Add photos (click camera icon or upload)
5. Click **"Add Item"** button

#### Method 2: Barcode Scan
1. Click **barcode scanner icon**
2. Scan product UPC/barcode
3. System auto-fills product details
4. Review and adjust as needed
5. Click **"Add Item"**

#### Method 3: Bulk Import
1. Go to **Settings → Import/Export**
2. Download CSV template
3. Fill in your data
4. Upload completed CSV
5. Review and confirm import

### Taking Photos (iPad/Mobile)
1. In Add/Edit modal, click **"Take Photo"**
2. Allow camera permissions
3. **Shutter button appears at bottom**
4. Take photo, review, retake if needed
5. Click **"Use Photo"** to attach

### Editing Items
1. Click on any item card
2. Click **"Edit"** button in detail modal
3. Make changes
4. Click **"Save Changes"**

### Deleting Items
1. Click on item card
2. Click **"Delete"** button
3. Confirm deletion

## Search & Filters

### Location
Top of inventory screen

### Search Bar
- Type to search across all fields
- Searches: name, manufacturer, model, serial number, caliber
- Real-time results

### Filter Panel
Click **filter icon** to open

**Available Filters:**
- Category
- Manufacturer
- Caliber
- Condition
- Storage Location
- Price Range
- Date Added

**Actions:**
- Apply filters
- Clear all filters
- Save filter preset

### Saved Filters
1. Configure filters
2. Click **"Save Filter"**
3. Name your filter preset
4. Access from **"Saved Filters"** dropdown

## Bulk Operations

### Location
Select multiple items → toolbar appears at bottom

### Selecting Items
- Click checkbox on item cards
- Or: Click **"Select All"** in toolbar

### Available Actions
- **Bulk Edit**: Change common fields
- **Bulk Delete**: Remove multiple items
- **Bulk Export**: Export selected items
- **Bulk Print Labels**: Print labels for selected items
- **Change Location**: Move items to new location
- **Change Status**: Update status for all selected

### How to Use
1. Select items (checkboxes)
2. Click action button in toolbar
3. Make changes in modal
4. Confirm action

## Import/Export

### Location
**Settings → Import/Export** or **Dashboard → Export button**

### Exporting Data

#### CSV Export
1. Click **"Export to CSV"**
2. Choose: All items or filtered items
3. File downloads automatically

#### PDF Report
1. Click **"Generate PDF Report"**
2. Select report type (inventory, valuation, etc.)
3. Customize options
4. Click **"Generate"**

#### Excel Export
1. Click **"Export to Excel"**
2. Includes all data and formatting
3. Multiple sheets for categories

### Importing Data

#### CSV Import
1. Click **"Import from CSV"**
2. Download template (if first time)
3. Upload your CSV file
4. Map columns (if needed)
5. Preview data
6. Click **"Import"**

#### Excel Import
1. Click **"Import from Excel"**
2. Upload .xlsx file
3. Select sheet to import
4. Map columns
5. Confirm import

## Barcode & Labels

### Barcode Scanning

#### Mobile Camera Scanner
1. Click **camera/barcode icon**
2. Point at barcode
3. Auto-scans when detected
4. Product info loads automatically

#### Batch Scanner
1. Go to **Tools → Batch Scanner**
2. Scan multiple items in sequence
3. Review scanned items
4. Bulk add to inventory

### Generating Labels

#### Single Item Label
1. Open item detail
2. Click **"Print Label"**
3. Choose label format (QR, barcode, detailed)
4. Print or save PDF

#### Bulk Labels
1. Select multiple items
2. Click **"Print Labels"** in toolbar
3. Choose format
4. Generate PDF with all labels

### Label Formats
- **QR Code**: Quick scan with location
- **Barcode**: Standard UPC/barcode
- **Detailed**: Full item info
- **Small**: Compact for small items

## Mobile Features

### Location
Available on mobile devices (iOS/Android)

### Install as App (PWA)
1. Open in Safari/Chrome
2. Tap **Share** button
3. Select **"Add to Home Screen"**
4. App icon appears on home screen

### Mobile-Specific Features

#### Pull to Refresh
- Pull down on inventory list
- Syncs latest data

#### Swipe Actions
- Swipe left on item: Delete
- Swipe right on item: Edit

#### Offline Mode
- Works without internet
- Syncs when connection restored
- Offline indicator shows status

#### Camera Integration
- Quick photo capture
- Barcode scanning
- Document scanning

#### Biometric Login
- Face ID (iOS)
- Touch ID (iOS)
- Fingerprint (Android)

**Setup:**
1. Go to **Settings → Security**
2. Enable **"Biometric Login"**
3. Authenticate once
4. Future logins use biometric

## Troubleshooting

### Common Issues

#### Photos Not Displaying
- Check internet connection
- Clear browser cache
- Try different browser

#### Camera Not Working (iPad)
- Allow camera permissions in Settings
- Close and reopen app
- Ensure good lighting

#### Items Not Saving
- Check required fields (marked with *)
- Verify internet connection
- Check error message for details

#### Sync Issues
- Pull to refresh
- Check **Sync Status** in settings
- View **Sync Queue** for pending items

### Getting Help

#### In-App Support
1. Go to **Settings → Help & Support**
2. Search knowledge base
3. Submit support ticket
4. View system status

#### Error Reports
- Errors automatically logged
- Admins can view in Error Analytics
- Export logs for support team

#### Contact Support
- Email: support@calibervault.com
- In-app chat (Pro/Team/Enterprise)
- Response time: 24-48 hours
