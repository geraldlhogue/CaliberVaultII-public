# Arsenal Command - User Guide

## Getting Started

### 1. Account Creation
1. Click "Sign Up" in the top right
2. Enter your email and password
3. Verify your email address
4. You're automatically in Trial Mode with 50 free transactions

### 2. Trial Mode
- **50 Free Transactions**: Add, edit, or delete items
- **Full Feature Access**: All features available during trial
- **Usage Tracking**: Banner shows remaining transactions
- **Upgrade Anytime**: Click "Upgrade" to unlock unlimited access

## Main Features

### Adding Items
1. Click "Add Item" button
2. Select category (Firearms, Optics, Ammunition, etc.)
3. Fill in required fields (category-specific)
4. Upload images (optional)
5. Click "Save"

**Quick Add via Barcode**:
1. Click "Scan Barcode"
2. Point camera at barcode
3. System auto-fills product details
4. Review and save

### Managing Inventory
- **Search**: Type in search bar to filter items
- **Filter**: Use advanced filters (price, date, caliber, etc.)
- **Sort**: Click column headers to sort
- **Bulk Actions**: Select multiple items for batch operations

### Categories
Each category has specific fields:
- **Firearms**: Serial number, caliber, barrel length, action type
- **Optics**: Magnification, objective diameter, reticle type
- **Ammunition**: Caliber, bullet type, quantity, grain weight
- **Suppressors**: Thread pitch, material, decibel reduction
- **Reloading**: Powder type, primer type, bullet weight
- **Accessories**: Type, compatibility, condition

## Troubleshooting

### Can't Add Items?
- Check if trial limit reached
- Verify internet connection
- Ensure all required fields filled

### Barcode Not Working?
- Check camera permissions
- Ensure good lighting
- Try manual entry if scan fails

### Sync Issues?
- Check internet connection
- Refresh the page
- Check sync status indicator
