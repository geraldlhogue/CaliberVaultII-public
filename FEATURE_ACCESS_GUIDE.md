## ğŸ†˜ Troubleshooting

### Can't Add Items?
1. Check if you're logged in (cloud features require account)
2. Verify database migrations are run (see DIAGNOSTIC_SUMMARY.md)
3. Check browser console for errors
4. Try refreshing the page

### Features Not Showing?
1. Some features require Pro/Enterprise subscription
2. Check your subscription level in Pricing tab
3. Ensure you're logged in for cloud features

### Sync Issues?
1. Check internet connection
2. Look for sync status in top left
3. Try manual sync from Backup tab
4. Check browser console for errors
